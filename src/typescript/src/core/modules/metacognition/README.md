# metacognition (META)

## Cosa fa
Auto-osservazione del sistema: confidence, coherence, power governance.

- **Meta Kernel**: Governance del "potere" del sistema (215 discipline)
- **Monitor**: Valutazione confidence e incertezza
- **Total System**: Orchestrazione parallela inconscia

## Cosa NON fa
- Non modifica autonomamente il comportamento
- Non "pensa" in senso umano
- Non ha agency propria

## Dove viene usato nel flusso
**Integrato in PLAN e prima di ACT** - Osservazione del processo.

```
PLAN → [metacognition] → Confidence OK? → ACT
               ↓ no
          Richiedi chiarimento
```

## File principali
- `kernel.ts` - Meta kernel (215 discipline, power envelope)
- `monitor.ts` - Metacognitive monitor
- `disciplines.ts` - Sintesi discipline
- `total_system.ts` - Total system orchestrator
